Remarks

Use case for this class. Dictionary can not be serialized in the unity inspector but simple cs classes can be serialized. Lists of serialized simple cs classes can be set up to look like a dictionary in the unity inspector . However Dictionaries are more performant for frequent random access than a lists. This class converts a list of serialized object to a dictionary and lists back.
